# Sydney Trains Trackwork Calendar

View alerts in Google Calendar for when buses replace trains.

![Monthly calendar view](./images/1.png)

For T1 Western Line, subscribe to this URL: https://raw.githubusercontent.com/dchenz/sydney-trackwork-calendar/refs/heads/ics/sydneytrains/WST

Alerts are added to the [`ics` branch](https://github.com/dchenz/sydney-trackwork-calendar/tree/ics) and Google Calendar fetches new data periodically.
